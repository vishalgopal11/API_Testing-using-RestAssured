# REST Assured API Automation Framework

## Overview
This project demonstrates API automation testing using REST Assured with Java and TestNG. The framework validates API responses, headers, and JSON schema.

## Features
- Request & Response validation
- Authentication handling
- JSON schema validation
- Reusable API utilities

## Tech Stack
Java  
REST Assured  
TestNG  
Maven  

## Sample API Tests
- GET Users API
- POST Create User API
- PUT Update User API
- DELETE User API

## How to Run Tests
mvn clean test
